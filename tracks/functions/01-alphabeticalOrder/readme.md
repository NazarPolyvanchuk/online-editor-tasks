# alphabeticalOrder

Напишіть функцію JavaScript, яка приймає рядок із буквами, та повертає його в алфавітному порядку.

**Приклад:**
Приклад рядка: 'webmaster'
Очікуваний результат: 'abeemrstw'

```js
alphabeticalOrder('webmaster') // abeemrstw
```
